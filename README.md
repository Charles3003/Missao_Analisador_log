# Missao_Analisador_log
Software de leitura de arquivo de log.
Para realizar a exexução do programa deverá instalar o Devexpress acessando: https://www.devexpress.com

Criar o banco de dados utilizando o script, que se encontra na pasta script_BD.

Após a criação do bando de dados, verá se alterado a connectionStrings, que se encontra no arquivo app.Config.
